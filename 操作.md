  解决方案：把项目复制到 Termux 的内部存储（支持 exec + symlink）再运行：

  # 1. 复制到 Termux 家目录
  cp -r /storage/emulated/0/源码/登录 ~/login-app

  # 2. 进入并安装
  cd ~/login-app
  npm install

  # 3. 启动开发服务器（自带热更新）
  npm run dev

  或者如果你有 Bun，也可以用（加载速度更快）：

  cd ~/login-app
  bun install
  bun run dev

  SD 卡本身有 noexec
  挂载限制，所有构建工具（esbuild、rollup）都需要可执行权限，所以代码源文件放 SD
  卡，运行环境放 ~ 是最佳方案。
